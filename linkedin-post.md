# LinkedIn Post - YAML Visualizer

---

🚀 **Fun day building something I'm passionate about!**

Instead of just talking about my love for DevOps and frontend development, I decided to build something that shows it. Meet **YAML Visualizer** - a tool that turns complex YAML configurations into interactive, visual graphs.

🔧 **What it does:**
- Visualizes Docker Compose, GitHub Actions, Kubernetes, and Helm templates
- Real-time graph updates as you edit
- Auto-detects YAML format
- Interactive nodes showing dependencies and relationships

💡 **Why I built this:**
Every DevOps engineer knows the pain of debugging nested YAML files. I wanted to create something minimal yet powerful - a tool that makes configuration files easier to understand at a glance.

🛠️ **Tech Stack:**
React + TypeScript + Vite + React Flow + Tailwind CSS + GSAP

The best part? It's open source and live!

🌐 Try it: https://yaseen-vm.github.io/YAML-Visualizer/
💻 GitHub: https://github.com/yaseen-vm/YAML-Visualizer

This project reminded me that with practice and passion, anyone can build anything. Whether you're a beginner or experienced developer, just start building - that's where the real learning happens.
